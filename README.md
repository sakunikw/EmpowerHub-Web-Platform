# EmpowerHub Web Platform

EmpowerHub is a web-based mentorship management platform designed to streamline mentorship activities between mentors and mentees.  
It provides a complete solution for user registration, mentor/mentee management, messaging, project tracking, and admin-level oversight.



##  Features

###  User Management
- Mentor & Mentee registration
- Profile updates
- CV uploads
- Secure login/logout system

###  Mentorâ€“Mentee Interaction
- Message sending and inbox system
- Manage mentorship details
- View mentor/mentee information

###  Admin Dashboard
- Manage users (add, edit, update, delete)
- View and delete user messages
- Send email to users
- Approve or remove mentors/mentees
- View all uploaded files & CVs

### Project & Tasks
- Users can upload projects
- Admin can view/manage projects

###  Communication Module
- PHPMailer integration for email sending
- User contact forms
- Admin notifications
